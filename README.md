# Bienvenue sur le frontend de Card'heure !
___
## Installation

- `git clone git@github.com:Card-heure/Front-FrontOffice.git`,
- `cd Front-FrontOffice`,
- `npm install`,
- `cp .env.example .env`,
- `npm run serve`
___
## Informations
Par defaut, le front office appel l'API via l'URL `http://localhost:8080/`. \
